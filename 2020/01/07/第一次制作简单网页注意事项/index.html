第一次制作简单网页注意事项

发表于 2020-01-07 | 分类于 学习

  1. 引入外部样式表

    <link herf=”css文件路径” rel=”stylesheet” />

  2. 别忘记删除默认的内外边距

    • margin: 0;
    • padding: 0;

  3. 别忘记多加注释，如导航栏开始和结束做标记

  4. 把某些元素放到一行如<a>,想清楚<a>本身在一行，是谁没让他在一行，如果是li的话，
    改li就行了

  5. markdown转义字符\

  6. 用”margin: 0 auto;”来调盒子内容，必须先给这个盒子宽高，否则没效果。因为盒子大小
    会随着内容改变而变化。

  7. font类，line类，text类的大多会继承。如：line-height

  8. background: url(../images/banner.jpg) no-repeat top;一定要注意：
    url里面不能加“”号。idea中会自动给你加上，切记要去掉。背景图片设置成功url中字体为蓝色

  9. 页面拉不动可以在body里加一个高度，让网页可以拉动。好看一点。
